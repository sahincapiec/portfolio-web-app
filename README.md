## Description

A simple portfolio NodeJS based web app that displays the profile image, name, some text with the experience, and a 5 tweet list of the user's Twitter timeline.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
